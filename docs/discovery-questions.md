# Discovery Questions for Board

1. Content scope: What exact sections are mandatory for v1 homepage (hero, KPI summary, positions table, alerts, transactions)?
2. Brand direction: Do we already have logo, color palette, and typography guidance to apply now?
3. Navigation: Should we build top-level nav now for Lending, Pendle PT, and Strategies, or keep single-page until data wiring starts?
4. Data inputs: Which chains/protocols must be supported first and will data be entered manually or imported from wallets/APIs?
5. Authentication: Do we need login/wallet auth in v1, or is this anonymous/local-only initially?
6. Hosting target: Preferred deployment platform for first preview (Vercel, Netlify, GitHub Pages, self-hosted)?
7. Analytics/privacy: Should we include product analytics at launch, and if yes, which provider and privacy constraints?
