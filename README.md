# DeFi Portfolio Dashboard - Initial Homepage Scaffold

This repository now includes the first homepage scaffold for DEF-8.

## Run locally

Option 1 (Python):

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

Option 2 (Node):

```bash
npx serve . -l 4173
```

## Current scope

- Static homepage layout
- Placeholder KPI and strategy cards
- Starter docs for discovery questions and frontend skill recommendation
